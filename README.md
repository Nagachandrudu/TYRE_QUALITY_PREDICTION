# 🚗 Tyre Quality Prediction System

This project implements an **AI-based tyre inspection system** using **Deep Learning** to classify tyre images as **Good** or **Defective**.  
It is designed for **industrial quality inspection**, **academic projects**, and **research demonstrations**.

---

## 📌 Project Overview

Manual tyre inspection is time-consuming and error-prone.  
This system uses a **Convolutional Neural Network (CNN)** trained on tyre images to automatically assess tyre quality from images.

---

## ✨ Features

- Image-based tyre quality prediction
- Binary classification:  
  - **Good Tyre**  
  - **Defective Tyre**
- Confidence score visualization
- Prediction confidence graph
- Deep learning model using **TensorFlow**
- Easy-to-run Python implementation

---

## 🧠 Technologies Used

- Python 3.10  
- TensorFlow / Keras  
- NumPy  
- Matplotlib  
- Flask (for optional web interface)
